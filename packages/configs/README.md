# Configs

Common configs.

## Usage

### Eslint preset

Add to package.json:

```json
{
    "eslintConfig": {
        "extends": "./node_modules/@bewitching-eggs/configs/eslint/<browser | server>"
    }
}
```

### Prettier preset

Add to package.json:

```json
{
   "prettier": "@bewitching-eggs/configs/prettier"
}
```
